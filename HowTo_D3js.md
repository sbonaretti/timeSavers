# How to: D3js

D3js provides two different kinds of method:  
- To transform data  
- To draw on webpage  

## Installation  
- Download the file from the website and then include `d3js` in the `.html` in the `script` section:  
  ```
  <script src="d3.js">  
  </script>
  ```
