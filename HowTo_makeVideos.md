# How To: Make videos  

## To record terminal  

### Asciinema: terminal in `.cast`
- Asciinema: https://asciinema.org/  
- Installation:    
  ```
  pip install asciinema
  ```  
- Start recording:  
  ```
  asciinema rec /path/filename.cast
  ```  
- Stop recording:   
  ```
  exit
  ```  
- Save: 
  ```
  crtl + c
  ```  
- Edit:  
  The `.cast` file is a `.json` file, so it can be opened with Atom and edited
